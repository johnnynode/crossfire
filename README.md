# 穿越火线(crossfire)

- 多年以前曾经玩过很长时间的穿越火线的网页特效模仿与代码展示 (jquery 版本)

### 克隆(clone)

- $ `git clone -b jquery git@github.com:johnnynode/crossfire.git --depth 1`

### 安装(install)

- $ `yarn install` or $ `npm i`

### 预览（preview) 开发模式

- $ `gulp server`

### 构建(build)

- $ `gulp build`

### 构建后的生产模式(build)

- $ `gulp build-server`
